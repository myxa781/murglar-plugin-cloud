
# Cloud Music Plugin for Murglar

English
--------
Plugin for Murglar providing access to cloud storage providers:
- Yandex Disk
- Dropbox
- Google Drive

Features:
- Browse folders
- Search audio files
- Stream music without full download
- Album and track navigation
- OAuth-based authentication

Security:
- No OAuth tokens, client secrets, API keys or user credentials are included in this repository.
- Users must create their own API applications where required and supply their own credentials.

Build:
- Android / Kotlin
- Gradle build system

See project license.

Русский
--------
Плагин для Murglar, предоставляющий доступ к облачным хранилищам:
- Яндекс.Диск
- Dropbox
- Google Drive

Возможности:
- Просмотр папок
- Поиск аудиофайлов
- Потоковое воспроизведение без полной загрузки
- Навигация по трекам и альбомам
- Авторизация через OAuth

Безопасность:
- В репозитории отсутствуют OAuth-токены, client secret, API-ключи и пользовательские учётные данные.
- Для работы пользователь должен использовать собственные приложения и собственные учётные данные.

Сборка:
- Android / Kotlin
- Gradle
